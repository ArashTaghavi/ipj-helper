# IPJ-Helper

# Usage:

Install with npm
```bash
npm i ipj-helper
```
CDN:
```html
<script src="https://unpkg.com/ipj-helper@1.0.0/index.js"></script>
```
Example:
```javascript
import { base64ToBlob } from 'ipj-helper'

const base64String='...'
<button onclick={()=> base64ToBlob(base64String) }> Open </button>
```
